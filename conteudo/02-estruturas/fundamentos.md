# Estruturas (struct)

Prof. Eduardo Ono

<br>

## Conceitos

> Uma `struct` é um meio na qual a linguagem C torna possível a criação de um tipo definido pelo usuário (programador). Basicamente, uma struct "encapsula" diferentes tipos primitivos de dados.
